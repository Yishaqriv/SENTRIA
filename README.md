# SENTRIA: AI-Alert Security Classification
SENTRIA es un proyecto de grado de Ingeniería de Sistemas en la Univeridad El Bosque enfocado en la optimización de la gestión de alertas de seguridad informática en PyMEs, integrando un SIEM open source (Wazuh) con un modelo LLM para filtrar falsos positivos y asistir en la generación de estrategias de protección.

El objetivo es reducir la fatiga por alertas, mejorar la priorización de incidentes y proporcionar una interfaz web interactiva que permita a los usuarios interpretar y gestionar alertas con ayuda de inteligencia artificial.
